# 模型示例参考

本页只供人工参考，可能过期。实际模型 ID 以服务商控制台、本地模型服务页面或 `/models` 返回结果为准。

重要约定：

- CLI 不读取本页。
- provider 不使用本页作为 fallback。
- 运行逻辑不会自动替用户选择模型。
- 生产配置里的 `main.model` 默认保持空字符串。

常见来源：

| 来源 | 获取方式 |
| --- | --- |
| 本地 OpenAI-compatible 服务 | 本地服务管理页面或 `/v1/models` |
| 阿里云百炼 / DashScope | 控制台或 compatible-mode `/models` |
| DeepSeek | 控制台或 `/models` |
| Moonshot Kimi | 控制台或 `/models` |
| 智谱 GLM | 控制台或 OpenAI-compatible `/models` |
| 硅基流动 | 控制台或 `/models` |

写入方式：

```powershell
mini-agent models use --profile <profile> --model "<模型ID>"
mini-agent config check --profile <profile>
```
